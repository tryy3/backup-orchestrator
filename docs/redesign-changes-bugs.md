# Design problems
We recently did a redesign but some views/pages got missed or feel a bit out of place that need to be updated.
Most of the effort was done on the Fleet Overrview page, so take inspiration from there when looking at how to improve the design problems below

## Plans list view
On the plans view the name of the backup plan has "text-blue-700" which feels very weird compared to the rest of the page.
Not entirely which colour would be suitable but at least not straight blue with a dark background

### Buttons
The buttons on this page also feels a bit out of the place, white/blue buttons doesn't follow the rest of the theme.

## Plans details view
This page missed a lot of the new changes and should probably get updated, a lot of cards have white backgrounds, buttons don't follow the patterns and so on.

## Repositories list view
Similarly to plans view the name of repositories are using text-gray-900, on a dark background it is very hard to see anything here

## Repository Edit Modal
This one probably got forgotten, when we are in the repository list view and click "edit" on a row the whole modal is white and just feel out of the place

### Modals overall
It seems most modals have the same issue, found similar design when looking at creating new scripts.
my guess we need to check all of the modals

## Dropdown fields
It looks like we might want to update dropdown fields too, when looking at the snapshot page the dropdown have white background with slight gray text making them very hard to read.
Either customize or maybe make custom dropdown that follow the same theme.
